## AnRail  區塊鏈寵物醫療紀錄系統
為飼主提供透明且安全的雲端寵物病歷App


####    環境要求： 

* Hyperledger fabric 2.2 LTS

* Ubuntu 22.04

* docker & docker-compose

* Node.js

#### Usage：
* 部屬網路 

    `cd network/` 進入 network 資料夾 <br>

    `./build.sh` 執行部屬腳本

* 安裝鏈碼

    `cd script/` 進入 script 資料夾 <br>

    `./chaincode-deploy` 執行鏈碼安裝腳本
